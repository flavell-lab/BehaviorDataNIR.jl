BehaviorDataNIR.jl Documentation

The BehaviorDataNIR.jl package provides a collection of utilities for computing behavior information from NIR recordings.

  • Locomotion API
    • Velocity computation
    • Stage data
  • Worm Posture API
    • Spline computation
    • Curvature computation
  • Feeding import API
  • Data IO API
    • Read data from NIR video and tracking network
    • Write video
    • Merge data from multiple datasets with the same animal
  • Confocal to NIR timesyncing API
  • Utilities API
    • Angle utilities
    • Other utilities
